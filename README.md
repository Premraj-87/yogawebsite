# Yoga Website

A modern, responsive yoga website built with React, TailwindCSS, and modern UI principles, designed to provide a calm and smooth user experience for yoga enthusiasts.

---

## Live Demo

https://yogawebsite-one.vercel.app/

---

## Features

- Clean and modern landing page design  
- Fully responsive layout for all devices  
- Smooth UI with consistent spacing and typography  
- Optimized React components for performance  
- Reusable component-based architecture  
- Minimal and calm wellness-focused design  
- Structured sections for classes, about, and contact  

---

## Tech Stack

React.js  
Tailwind CSS  
JavaScript (ES6+)  
Vercel (Deployment)  

---

## Project Overview

This project represents a yoga studio or wellness brand website focused on simplicity, clarity, and a peaceful user experience. The layout is designed for easy navigation and smooth user interaction.
