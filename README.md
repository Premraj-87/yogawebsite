🧘 Yoga Website

A modern, responsive yoga website built with React, TailwindCSS, and modern UI principles to provide a calm and smooth user experience for yoga enthusiasts.

 Live Demo

 https://yogawebsite-one.vercel.app/

 Features
 Clean and modern landing page design
 Fully responsive (mobile, tablet, desktop)
 Smooth UI with elegant spacing and typography
 Fast performance with optimized React components
 Reusable component-based architecture
 Modern aesthetic focused on calm & wellness vibe
 Sections for classes, about, and contact


Tech Stack
React.js
Tailwind CSS
JavaScript (ES6+)
Vercel (Deployment)
Project Overview

This project is designed to represent a yoga studio / wellness brand website, focusing on simplicity, clarity, and a peaceful user experience. The layout is optimized for engagement and easy navigation.

Getting Started
git clone https://github.com/your-username/yogawebsite.git
cd yogawebsite
npm install
npm run dev

 Purpose

Built to practice modern frontend development and create a real-world wellness website UI with focus on design consistency, responsiveness, and user experience.
